# azure.synapse.tools

## What is supported
The deployment of these objects:  
- Workspace instance
- dataset
- dataflow
- integration runtime
- linked service
- pipeline
- KQL script *
- SQL script *
- notebook *

> \* via RestAPI only

## What is NOT yet supported
The deployment of these objects:
- credential
- Spark job definition
